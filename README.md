# Patient Data Fetch System

## Description
This is a Python-based project that fetches user data from an API and displays it in a structured format.

## Features
- Fetch data using REST API
- Input validation
- Loop-based interaction
- JSON data parsing

## Technologies Used
- Python
- Requests library
- REST API

## How it works
